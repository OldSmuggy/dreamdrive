The Blender Toolkit skill empowers Claude Code to interact directly with Blender 4.0+, enabling developers and 3D artists to automate complex tasks like primitive generation, PBR material assignment, and mesh modifiers. It features a sophisticated animation retargeting system that utilizes fuzzy bone matching and a two-phase confirmation workflow to seamlessly apply Mixamo animations to custom character rigs. By bridging the gap between Claude's reasoning and Blender's 3D engine via a localized WebSocket server, it transforms manual 3D modeling and animation pipelines into programmable, high-efficiency workflows.
